Tạo link cài đặt iOS (.plist) từ file mẫu online

* .plist mẫu sẽ được lấy tự động từ: https://github.com/nhut990/files/raw/refs/heads/main/hinh/Test_taocaidat.plist
* Sau khi tải file .plist, bạn cần upload lên host HTTPS của bạn (github.io, firebase, vps...)
* Link cài đặt itms-services://... chỉ hoạt động trên iPhone/iPad.
* File .plist mẫu dùng đúng 4 cụm từ: Nhập Link iPA, Nhập Bundle ID, Nhập Phiên Bản, Nhập Tên App
